# Personal Website of Glendc

The source code of the Glendc website.

https://www.glendc.com

## Build Locally

Install Zola first:

> <https://www.getzola.org/documentation/getting-started/installation/>

Build:

```
$ zola build -o docs
```

Or serve:

```
$ zola serve --interface 127.0.0.1 --port 8080
```
